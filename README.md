# pdfToCsv
Download pdf file and convert table of pdf file to csv file, finally send by email.

* Download pdf file and convert table of pdf file to csv file, finally send by email.

    npm install 
    
    node server.js

Server Run 

    http://localhost:3000

Main Function

    (1) Get User Input => Date, Total, Email
    (2) Download Pdf File
    (3) Convert Pdf to Csv File ( Only Table )
    (4) Csv send by Email
